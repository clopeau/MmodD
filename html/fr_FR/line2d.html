Name

line2d — Type declaration

Calling Sequence

   line2d(varargin)

Parameters

varargin :

if string then it's the line2d 's identity and if matrix then it's line2d 's point coordinates

Description

line2d returns an empty list if varagin is a string or a list that contains : Id : identity Coor : coordinates matrix Seg : segment matrix

Examples

a=line2d('name')
a.Id
b=line2d([1 1;2 2; 5 0;9 9])
b.Id='my_line2d'
b.Coor
b.Seg
